I'm a UMass Amherst building projects across data, AI, and full-stack web development.

I like building things that are practical, fast, and useful. I’m especially interested in the space where software meets business: tools that help people make better decisions, move quicker, and solve real problems.

Before tech, I competed as a national swimmer, so discipline, speed, and pressure have always been part of how I work. I’m bringing that same mindset into the products I build now — focused, competitive, and always trying to improve.

### 🦈 Play Lane Shark — my latest project

> A fast-paced swim-lane arcade game inspired by competitive swimming. Race a shark through 5 ocean lanes, collect fish, dodge jellyfish, and hit the finish line.  
> Built with **zero dependencies** — pure HTML, CSS, and JavaScript.
